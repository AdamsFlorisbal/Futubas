## :information_source: Como usar

Para executar esta aplicação, você precisa do e [Yarn](https://yarnpkg.com/) ou maior instalados. Comandos:

```bash

# Go into the repository
$ cd react-login

# Install dependencies
$ yarn

# Install dependencies
$ npm install

# Run the app
$ yarn start

# Run the app
$ npm start
```
